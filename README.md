# dobro-store

## Requirements
1. JDK 11
2. Maven 3.8.2

## Quick Start
### For development
1. Установить Docker
2. Перейти в корневую директорию проекта (./dobro-store)
3. mvn clean install
4. Вызвать в комадной строке docker-compose up --build
5. Swagger - http://localhost:8080/swagger-ui/index.html
